Skip List

This memory-secure skip-list data structure combines the speed of a binary search tree with the conveinience of a linked-list. Developed in C++, the skip list uses multiple "layers" to traverse the list in an average O(log(n)) time.

This project was completed over three days in May 2021. It was developed as a part of coursework.


Keylogger

This simple keylogger, when running in the background, records a user's keypresses and saves them to a text file.

This project was developed in an afternoon in July 2021. I developed it to experience new technologies and practice my coding abilities.



SenateBot, an Encryption Discord Bot

This bot, developed for use on Discord servers, takes in a user's message and performs a simple Caesar-cypher encryption and decryption on it. It also allows users to translate their text into morse code.

Developed with a group as a part of HowdyHack, a hackathon that took place in September 2020.


Each of these projects - including this website - taught me something new and valuable about fundamental computer science topics.
